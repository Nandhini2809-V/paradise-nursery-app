# Paradise Nursery

## Project Description
Paradise Nursery is a React-based e-commerce web application for selling houseplants online. 

This application allows users to browse different categories of plants, add them to the shopping cart, update quantities, and view the total price. 

## Features
- Landing page with background image
- Product listing page with 6 different houseplants
- Add to Cart functionality using Redux
- Shopping Cart page with quantity update and delete option
- About Us page
- Responsive navigation bar

## Technologies Used
- React.js
- Redux Toolkit
- React Router DOM
- CSS
